MAX Question Answering Chatbot Demo

The MAX Questioning Answering model is a deep learning algorithm that attempts to answer questions based on a piece of text. For this demo, the Biology textbook by M. Clark, M. Douglas, and J. Choi was selected. The chatbot below is an interface to interact with the algorithm.
